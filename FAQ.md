# Frequently Asked Questions (FAQ)

### 1. Where does the data come from?
The data is a compilation of various international surveys. See `SOURCES.md` for details.

### 2. Can I use this for commercial purposes?
Yes, the dataset is under CC0 1.0, which allows commercial use.

### 3. How should I cite this?
Refer to the `CITATION.cff` file or the Citation section in `README.md`.

### 4. How often is the data updated?
We aim for an annual update cycle to incorporate the latest survey results.
