# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within this repository, please send an email to juanmoises.delaserna@unir.net. All security vulnerabilities will be promptly addressed.

## Supported Versions
| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Scope
This policy covers the data files, analysis scripts, and CI/CD workflows contained in this repository.
