## Resumen
Este conjunto de datos recopila datos cuantitativos de encuestas internacionales (2018-2024) sobre las actitudes de los investigadores académicos hacia la ciencia abierta y el intercambio de datos. Incluye cinco dimensiones analíticas clave: (1) Voluntad de compartir datos por disciplina; (2) Tasas de conocimiento y uso de repositorios (Zenodo, Dataverse, OSF, Figshare, Dryad, GitHub); (3) Barreras percibidas (problemas legales/éticos, falta de reconocimiento, etc.).
