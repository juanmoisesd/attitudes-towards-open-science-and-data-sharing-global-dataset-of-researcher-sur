# Resumen (Abstract)

Este conjunto de datos recopila datos cuantitativos de encuestas internacionales realizadas entre 2018 y 2024 sobre las actitudes de los investigadores académicos hacia la ciencia abierta y el intercambio de datos. El estudio abarca cinco dimensiones analíticas fundamentales: la voluntad de compartir datos según la disciplina académica; el conocimiento y la frecuencia de uso de repositorios de datos (como Zenodo, Dataverse, OSF, Figshare, Dryad y GitHub); las barreras percibidas para el intercambio de datos (incluyendo aspectos legales, éticos y de reconocimiento); los incentivos necesarios para fomentar prácticas abiertas; y la evolución temporal de estas actitudes. Los resultados proporcionan una base empírica para el desarrollo de políticas de ciencia abierta y estrategias de gestión de datos de investigación a nivel global.
