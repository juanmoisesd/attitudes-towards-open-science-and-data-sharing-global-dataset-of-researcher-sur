# Governance Model

## Roles
- **Project Lead:** Juan Moisés de la Serna Tuya (Decision maker).
- **Maintainers:** Responsible for code review and infrastructure.
- **Contributors:** Community members providing data or code improvements.

## Decision Making
Major changes to the dataset schema or project scope are decided by the Project Lead after community consultation via GitHub Issues.

## Transparency
All changes are tracked via git commits. Releases are documented and signed.
